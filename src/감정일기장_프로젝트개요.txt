        - Home  페이지: 
            - 앱에 접속하면 처음 만나는 페이지
            - 경로설정 => "/"
            - 상단영역 (월단위 일기 조회, 좌우버튼으로 날짜 이동)
            - 리스트 영역(날짜기준으로 최신,오래된 순, 새로운 일기 추가 기능, new 페이지로 이동)
        - New   페이지: 
            - 페이지 경로 = "/new"
            - 상단영역(뒤로가기 버튼, 새일기 쓰기 제목)
                :날짜 입력, 감정 이미지 선택, 일기 작성 폼
            - 하단 영역
                :취소하기, 작성완료 버튼
        - Diary 페이지
            - 페이지 경로 /dairy/(일기)id
            - 작성한 일기 상세히 조회하는 페이지
            - 하단영역
                :뒤로가기,수정하기 버튼
        - Edit  페이지
            - 페이지 경로 /edit/(일기)id
            - 작성한 일기 수정하는 페이지
            - 하단영역
                :수정완료, 뒤로가기 버튼
